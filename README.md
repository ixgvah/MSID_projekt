## ❗️Cel projektu 

Celem projektu jest zapoznanie studentów z działaniem wybranych algorytmów metaheurystycznych poprzez ich implementację oraz zastosowanie do rozwiązania prostego problemu optymalizacji.
W projekcie należy:
1	wybrać prosty problem optymalizacji kombinatorycznej,
2	zaimplementować kilka algorytmów metaheurystycznych,
3	porównać ich skuteczność.


## 🖌 Instalacja

1. Sklonuj repozytorium:
```bash
git clone https://github.com/ixgvah/MSID_projekt.git
cd MSID_projekt

2. Utwórz i aktywuj wirtualne środowisko
python -m venv venv
# Linux / macOS
source venv/bin/activate
# Windows
venv\Scripts\activate

3. Uruchom projekt
  python main.py
```
